# 全国招聘平台地图与新增爬虫规划

## 目标

这份文档用于统一回答三个问题：

1. 目前市面上有哪些靠谱且值得接入的招聘平台。
2. 当前项目已经覆盖了哪些平台，还缺哪些高价值来源。
3. 后续新增爬虫应该按什么顺序推进，避免盲目铺站点。

这里不把“平台”只理解为商业招聘 App，而是分成四大类：

1. 国家级与官方背景平台。
2. 全国性综合招聘平台。
3. 地方人才网与公共就业网。
4. 垂直细分行业与应届生平台。

## 二、国家级与官方背景平台

这类平台的价值不只是“多一个来源”，而是具备更高的真实性、审核强度和校验价值。

### A. 已覆盖

1. 国聘

### B. 尚未覆盖但强烈建议纳入目录的官方平台

1. 国家大学生就业服务平台 24365
定位：应届生、在校生、实习、校招。
价值：教育部主办，校招与应届资源很强，适合作为校招正规来源与官网入口验证器。
建议：优先做企业专题、校招专题和官方投递链接采集，不必一开始追求全站职位抓取。

2. 中国公共招聘网
定位：公共就业服务、基层项目、事业单位、社会招聘。
价值：人社部背景，适合补公共就业体系和官方岗位发布。
建议：优先做公告与岗位目录，不急于做深层反复翻页。

3. 就业在线
定位：官方聚合搜索入口。
价值：更适合作为平台发现器、官方来源索引器，而不是主 jobs 表的大规模职位源。
建议：优先做“平台目录 + 外链映射 + 聚合入口整理”。

### C. 相关官方体系，建议作为地方入口总线

1. 各省人社厅就业服务平台
2. 各省公共招聘网
3. 各省人才服务中心官网

这些站点不一定都适合作为第一批职位主来源，但非常适合作为：

1. 官方岗位真伪校验源。
2. 地方国企与事业单位入口发现器。
3. 地方人才网种子目录。

## 三、全国性综合招聘平台

这类平台的共同特点是岗位量大、覆盖全国、适合撑起 jobs 主表规模。

### A. 已覆盖

1. Boss直聘
2. 智联招聘
3. 前程无忧 51job
4. 猎聘
5. 拉勾
6. 实习僧

### B. 尚未覆盖但值得纳入平台目录的综合来源

1. 58同城 / 赶集直招
定位：蓝领、兼职、本地服务、物流、零售、餐饮。
价值：可显著补足当前项目在蓝领、本地生活服务、基层岗位上的空白。
风险：信息质量差异大，诈骗和重复信息相对更多。
建议：作为单独板块处理，不建议直接和高质量白领岗位混排；先做“本地服务/蓝领”二级来源。

2. 鱼泡网
定位：建筑、装修、工地、技工、蓝领。
价值：若后续要覆盖建筑与技能工种，这类站点比白领平台更有效。
建议：单独建蓝领标签，不直接混入普通校招/白领分析页。

3. 店长直聘 / 招才猫直聘 等行业型综合平台
定位：零售、餐饮、门店、生活服务。
价值：补足当前服务业岗位。
建议：优先记录到来源目录，是否接爬虫视后续岗位结构需要决定。

## 四、地方人才网与公共就业网

这是“全国铺开”时真正容易被忽略，但又最能补地方岗位差异化的数据层。

### A. 本地官方平台优先级判断原则

地方平台里最值得优先接的是三类：

1. 省级公共就业网 / 省级人才网。
2. 市级官方人才网 / 公共就业网。
3. 招才引智平台 / 区属国企招聘平台。

这些平台尤其适合：

1. 青岛、山东本地岗位补全。
2. 地方国企、城投、地铁、医院、学校、事业单位公告。
3. 商业平台没有同步或同步很慢的地方岗位。

### B. 青岛与山东优先种子

这是当前项目最值得优先整理和后续接爬虫的一组地方来源。

1. 青岛人才网
类型：政府官方。
价值：青岛公务员、事业单位、国企、重点企业招聘信息的高可信入口。
优先级：高。

2. 青岛公共就业网
类型：政府公益。
价值：适合高校毕业生、技能人才、公共就业岗位。
优先级：高。

3. 齐鲁人才网
类型：省级市场化龙头。
价值：覆盖山东 17 地市，本地制造业、消费品、家电、啤酒、供应链企业多。
优先级：高。

4. 青青岛人才网
类型：本地市场化平台。
价值：适合补青岛本地中小企业和生活服务岗位。
优先级：中。

5. 各区市招才引智平台
类型：区级官方。
价值：适合抓区属国企、区属事业单位、补贴项目、紧缺人才公告。
优先级：中。

### C. 可扩展到全国的地方平台目录模板

由于全国地方人才网数量很多，后续建议按省市两级建立目录，而不是试图一口气为每个城市都手工做深适配。

建议目录字段：

1. region_level：省级 / 市级 / 区县级。
2. province_name：省份。
3. city_name：城市。
4. platform_name：平台名称。
5. platform_type：官方人才网 / 公共就业网 / 招才引智 / 市场化地方站。
6. official_site_url：站点首页。
7. jobs_url：职位或公告列表页。
8. suitable_for：社招 / 校招 / 国企 / 蓝领 / 事业单位。
9. crawl_priority：high / medium / low。
10. notes：是否适合接口、是否有 JSON、是否必须浏览器。

### D. 全国地方平台建议按区域逐步接入

建议优先顺序：

1. 山东：青岛、济南、烟台、潍坊、临沂。
2. 京津冀：北京、天津、石家庄。
3. 长三角：上海、杭州、南京、苏州、宁波、无锡。
4. 珠三角：广州、深圳、佛山、东莞、珠海。
5. 中西部重点：成都、武汉、西安、重庆、长沙、郑州。

原因：

1. 这些城市本身岗位密度高。
2. 与当前全国扩容路线里的城市集群一致。
3. 更容易快速放大城市覆盖与岗位结构差异。

### A. 已覆盖

### B. 尚未覆盖但值得纳入平台目录或专题板块的垂直来源

1. 牛客校招专题 / 企业页
定位：校招、实习、企业批次、官网投递入口。
价值：更适合作为“企业专题与官网入口发现器”，不建议作为 jobs 主表大体量来源。
优先级：高。

2. 应届生求职网
定位：校招、宣讲会、企业招聘专题。
价值：适合补校园招聘、企业批次、邮箱投递和官网入口。
优先级：高。

3. 丁香园
定位：医疗、医药、医院岗位。
价值：若后续要覆盖医疗教育类岗位，是很强的垂直补充。
优先级：中。

4. 教师人才网 / 高校人才网类平台
定位：教育、教师、高校岗位。
价值：补学校、教师、教辅类岗位。
优先级：中。

5. 一览英才网
定位：工程、制造、建筑、环保、电力等专业岗位。
价值：补当前技术岗位之外的工程产业链岗位。
优先级：中。

6. 医药英才网 / 建筑英才网 / 化工英才网
定位：行业垂直岗位。
价值：适合做行业板块，而不是当前第一阶段全国大盘扩容主来源。
优先级：中低。

## 六、平台全景分层结论

如果从“靠谱 + 有用 + 值得维护”的角度看，后续平台池建议分成四层。

1. Boss直聘
2. 智联招聘
3. 前程无忧 51job
4. 猎聘
5. 实习僧
6. 国聘
7. 拉勾

目标：

1. 冲在库规模。
2. 做全国分析。
3. 快速补城市覆盖。

### 第二层：官方与地方正规来源

1. 24365 国家大学生就业服务平台

3. 就业在线
4. 青岛人才网

目标：

1. 提升真实性。
2. 补地方岗位。

### 第三层：企业专题与官网入口来源

## 八、新增爬虫整理顺序建议

### 阶段 1：把官方与地方目录先建出来

建议先做目录型整理，而不是急着每个平台都写完深度爬虫。

先做：

1. 新建平台地图数据文档。
2. 整理平台名称、定位、区域、优先级、备注。
3. 给每个平台打上“适合 jobs 主表 / 适合企业专题 / 适合官方校验”的标签。

### 阶段 2：优先新增 3 个高价值来源

建议新增顺序：

1. 青岛人才网
2. 齐鲁人才网
3. 24365 国家大学生就业服务平台

原因：

1. 一个补本地官方。
2. 一个补山东市场化。
3. 一个补全国官方校招。

### 阶段 3：补企业专题类来源

建议新增顺序：

1. 牛客校招专题
2. 应届生求职网企业专题
3. 企业官网招聘站

原因：

1. 这批更适合直接增强名企与央国企专题。
2. 不一定马上推 jobs 主表，但会显著增强企业专题质量。

### 阶段 4：补公共就业与蓝领来源

建议新增顺序：

1. 中国公共招聘网
2. 青岛公共就业网
3. 58同城 / 赶集直招

原因：

1. 先补官方公共就业体系。
2. 再决定是否切入蓝领与生活服务岗位。

## 十、当前结论

如果从“尽量把靠谱平台先找全，再决定爬虫怎么补”的角度看，当前项目最值得继续推进的不是再盲目接更多商业站，而是：

1. 保持全国综合平台继续扩量。
2. 把国家级官方平台补进目录。
3. 把青岛与山东本地平台作为地方第一优先级。
4. 把牛客、应届生和企业官网作为专题型来源补进来。
5. 把 58 同城这类蓝领来源单独规划，避免污染现有分析口径。

可以把后续新增来源分成三条线同时推进：

1. jobs 主表扩量线：继续扩 Boss、智联、51job、猎聘、实习僧、国聘。
2. 官方正规来源线：24365、中国公共招聘网、青岛人才网、齐鲁人才网。
3. 企业专题来源线：牛客、应届生、企业官网招聘站。
